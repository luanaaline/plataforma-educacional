Perguntas Frequentes | EducAê

O que é o Educae? +
O Educae é uma plataforma educacional criada para facilitar o aprendizado, oferecendo conteúdos de qualidade, linguagem simples e ferramentas para estudantes e professores.
O Educae é gratuito? +
Sim! Parte do conteúdo é totalmente gratuito. Alguns materiais especiais podem estar disponíveis em planos premium.
Quais tipos de conteúdo posso encontrar? +
Você encontrará videoaulas, simulados, ferramenta de criação de redações e mapas mentais de várias matérias.
Como posso entrar em contato com o suporte? +
Basta enviar um e-mail para educae.mentoriae@gmail.com
Preciso criar uma conta? +
Sim, criar uma conta permite salvar seu progresso e acessar todos os recursos.
Os conteúdos do Educae são atualizados com que frequência? +
Atualizamos nossos materiais regularmente, incluindo novos mapas mentais, e simulados conforme surgem mudanças curriculares e novas demandas estudantis.
O Educae oferece simulados? +
Sim! Há simulados para diversas matérias, todos com correção automática para facilitar seu aprendizado.
O Educae possui materiais voltados ao ENEM? +
Sim! Temos resumos, exercícios, simulados e mapas mentais voltados especialmente para o ENEM e vestibulares.
