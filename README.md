# 💻 EducAê

Plataforma educacional interativa desenvolvida para apoiar estudantes em seus estudos, promovendo autonomia e aprendizado de forma dinâmica.

## 🚀 Funcionalidades

- 📺 Videoaulas
- 📝 Conteúdos de redação
- 🧠 Mapas mentais
- ❓ Quizzes interativos
- 📚 Recursos para estudo autônomo

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 📌 Objetivo

O EduCAE foi desenvolvido como Trabalho de Conclusão de Curso (TCC), com o objetivo de auxiliar estudantes no processo de aprendizagem, oferecendo uma plataforma acessível e interativa.

## ▶️ Como executar

1. Baixe ou clone este repositório  
2. Abra o arquivo `index.html` no navegador
